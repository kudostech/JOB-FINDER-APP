# 🚀 Job Finder App

A **simple Job Finder web app** built with **React, Vite, Tailwind CSS, and Axios**. This app fetches job listings from a dummy API, displays them cleanly, and allows users to **search for jobs by title** instantly.

---

## ✨ Features

✅ Fetch job listings using Axios  
✅ Search and filter jobs by title  
✅ Responsive, clean, modern UI with Tailwind CSS  
✅ Built with Vite for fast builds and hot reload  
✅ Ready for extension with real APIs or Firebase backend

---

## 🛠️ Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Data Fetching:** Axios
- **Deployment:** Vercel
